Godaddy密码及Password Hint丢失，傻瓜式找回2013

Posted on 2013-02-22 Edited on 2021-01-08 In Wordpress

回来一段时间，突然收到邮件，提示域名还有XX日到期，菊花一紧。这折腾一阵的域名，别没了。赶紧上godaddy上看看状态。但是思来想去，godaddy已经有N多时日没登了，账号密码全部忘记了，唯一记得的是注册邮箱。如过凑巧用户名或者customer number也记住的话，更好了，按照这个来找回密码。 1、先进首页，选择Forgot Password

2、继续重置密码，reset password，输入你看到的信息。点击黄色按钮 submit，提交。

3、你会收到一个提示，要求你查看邮箱，不要怀疑，赶紧打开邮箱查看。下面的红色的your authorization code is ： 不要管，后面点击链接后会自动填充。点击蓝色链接 click here

4、进入Account Assistance，右侧鉴权码自动填充，输入相关信息，用户名/customer number，输入新密码（新密码不能与之前使用的5个个老密码），重复输入。有人在password hint 这里迷失了，我就犯过这事。确实不知道是什么，当初的早就忘记了。其实不要紧，随便输入几个数字就好。我输入的是4个随即数字后，完成密码重新设置。提交，然后会看到提示，已经修改成功。

5、用新密码登入即可。 6、如果不知道customer number#，不要紧。看这里，同样能找回。

7、登陆邮箱查找，your log in information. 看灰色条目上信息，your customer number is：输入即可。

8、重复步骤1-5。 其实，我给客户打过电话，但是实在听得稀里糊涂。最后，才知道password hint 其实可以重新设置一个，忘记了，没关系。N久之前也弄过一次，具体过程忘记了，这次写在这里，给自己提个醒，不至于下次再慌慌张张。当然能够记得住账号密码，是最理想的状态。呵呵 ，如果记不住怎么办，不妨做个备忘录吧，存哪儿都行。 希望能帮到你，我的朋友。完。
